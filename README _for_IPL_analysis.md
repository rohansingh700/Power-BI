# Power-BI
Power bi Project on Uber and IPL 

🏏 IPL Data Analysis Dashboard (Power BI)
📌 Overview

This project is an interactive IPL (Indian Premier League) Data Analysis Dashboard built using Power BI. It provides insights into team performance, player statistics, match outcomes, and season trends.

The dashboard helps users explore IPL data visually and make data-driven observations.

📊 Features
📅 Season-wise analysis
🏆 Team performance comparison
👨‍🏏 Player statistics (runs, wickets, etc.)
📍 Venue-based insights
📈 Match trends and win analysis
🔍 Interactive filters and slicers
🛠️ Tools & Technologies Used
Power BI (.pbix)
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)
Data Visualization Techniques
📂 File Included
iplanalysis.pbix → Main Power BI dashboard file
🚀 How to Use
Download the .pbix file from this repository
Open it using Microsoft Power BI Desktop
Interact with filters, slicers, and visuals
Explore insights across different IPL seasons
